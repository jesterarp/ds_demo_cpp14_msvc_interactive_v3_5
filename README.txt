Data Structures & Algorithms Demo (v3.5 - clean baseline)
=========================================================
- MSVC v143 (VS2022), C++14, x64, Multi-Byte, /W3, /MTd (Debug), /MT (Release), /Zc:twoPhase-
- DO-178B/C discipline: strict single-return-per-function (no early returns)
- Minimal baseline: no logging (added later once build is clean)

Build:
1) Open ds_demo_cpp14_msvc_interactive_v3_5.sln in VS2022
2) Select Debug | x64 (or Release)
3) Build → Rebuild Solution
4) Ctrl+F5 to run
